本仓库专门用于存储洛雪音乐客户端的各种版本安装包，方便用户下载使用。

## 安装包列表

### Windows版本
- lx-music-win-Vx.x.x-XXX.exe (最新稳定版)
- lx-music-win-Vx.x.x-XXX.zip (便携版)

### macOS版本
