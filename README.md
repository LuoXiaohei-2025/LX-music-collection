# LX-music-collection

洛雪音乐音源及安装包收集整理项目

## 项目简介

本项目收集整理了洛雪音乐的音源和安装包，音源的可用性需要大家自行尝试。

## 重要说明

- 洛雪音乐官方电脑版仓库：https://github.com/lyswhut/lx-music-desktop
- 洛雪音乐官方手机版仓库：https://github.com/lyswhut/lx-music-mobile
- 本项目仅为音源收集整理，不保证所有音源的可用性
- 音源质量、稳定性和可用性可能随时变化
- 使用前请自行测试音源是否满足您的需求
